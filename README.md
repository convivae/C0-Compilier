# C0-Compilier
## 使用说明

接口如下：

```
Usage:
  cc0 [options] input [-o file]
or 
  cc0 [-h]
Options:
  -s        将输入的 c0 源代码翻译为文本汇编文件（.c0-->.s）
  -c        将输入的 c0 源代码翻译为二进制目标文件（.c0-->.o）
  -h        显示关于编译器使用的帮助
  -o file   输出到指定的文件 file
  -t        仅进行词法分析
  -a        将输入的文本汇编文件翻译为二进制目标文件(.s-->.o)
  -r        执行输入的二进制目标文件
  -d        将输入的二进制目标文件进行反汇编

不提供任何参数时，默认为 -h
提供 input 不提供 -o file 时，默认为 -o out
```

- 未初始化的非`const`变量，在栈上为其分配内存空间，初始值不确定
